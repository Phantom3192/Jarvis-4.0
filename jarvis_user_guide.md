# Welcome to Jarvis 👋

Jarvis is the AI assistant, game master, music player, and all-around toolbox for this server.
This guide covers everything you need to get started.

---

## 🤖 Talking to Jarvis

Just chat! You can:
- **@mention Jarvis** or **DM Jarvis** directly, or
- Use `/chat` if you prefer a slash command

Jarvis remembers your last couple of messages so it can follow a conversation, and it can
also **remember things about you long-term** — your name, where you're from, what you like —
so you don't have to repeat yourself every time. Just tell it naturally, e.g. *"call me Alex"*
or *"remember that I love hiking."*

Useful commands:
| Command | What it does |
|---|---|
| `/setmodel` | Choose which AI model replies to you (there's a fast one, a smart one, and one that can read images) |
| `/mymodel` | Check which model you're currently using |
| `/mymemory` | See everything Jarvis remembers about you |
| `/forgetme` | Wipe everything Jarvis remembers about you |
| `/clearhistory` | Clear your recent conversation |
| `/nickname` | Tell Jarvis what to call you |
| `/summary` | Get a quick summary of your recent conversation |
| `/dnd` | Toggle Do Not Disturb — quiets most of Jarvis's responses to you |
| `/remindme` | Set a DM reminder for later |
| `/myreminders` | View your active reminders |

You can also **send images** — Jarvis can look at and describe/answer questions about pictures
(depending on the model you've selected).

### Daily limit
Everyone gets **100 free AI messages a day**, resetting at **midnight UTC**. You'll get a
heads-up once you've used 80. If you run out, you can either wait for the reset or spend some
Jarvis Credits (see below) to reset your limit early with `/mylimit`.

## 🪙 Jarvis Credits (JC)

Jarvis has its own currency you can earn just by using the bot, and spend on perks and fun
extras.

### How to earn JC
| Action | Reward |
|---|---|
| Send your first message of the day | 50 JC |
| Being new here (one-time) | 50 JC |
| Chatting with the AI | 5 JC per reply (up to 100 JC/day) |
| Keeping a daily chat streak going | 200 JC at 7 days, 500 JC at 30 days |
| Getting a suggestion or bug report accepted | 50 JC |
| Referring a friend who redeems your code | 50 JC |

### How to spend JC
| Item | Cost |
|---|---|
| Reset your daily AI limit early | 50 JC |
| Save a broken counting-game streak | 25 JC |
| Get an extra chess hint | 15 JC |
| Open a Mystery Box (random 100–300 JC reward) | 200 JC |

### Commands
- `/balance` — check how much JC you have
- `/leaderboard` — see the top JC holders on the server
- `/streak` — check your daily chat streak
- `/shop` — browse what you can buy
- `/mysterybox` — open a mystery box
- `/invite` — get your personal referral code to share
- `/redeem` — use a friend's referral code (only works the very first time you interact with
  Jarvis)
- `/transferjc` — send JC to another user

## 🎮 Games

### Chess
Challenge someone with `/chess`. Make moves with `!move`, offer/accept a draw with `!draw`,
resign with `!resign`, or request to undo your last move with `!undo` (your opponent has to
agree). Stuck? Use `!hint` for an AI-suggested move — your first hint each turn is free, extra
ones cost JC. `!chessboard` redraws the current board if you've lost track of it.

### Mafia 🎭
A full social-deduction game for a group. Start a lobby with `/mafia`, and the host can pick
from 11 possible roles (Mafia, Detective, Doctor, Villager, Vigilante, Jester, Mayor, Serial
Killer, Bodyguard, Spy, Escort). Day and night phases move forward automatically once everyone's
acted, or when the timer runs out. You can write a "last will" that gets revealed if you're
eliminated, and Mafia members get a private chat to coordinate. Run `/mafguide` for the full
rules explainer before your first game.

### Hangman
Quick and simple — `/hangman` starts a round, guess letters to save your character before the
gallows fill in. `!stophangman` ends it early.

### Counting Game
A shared server game — take turns counting up one number at a time (nobody can count twice in
a row). Ask an admin to set up a counting channel with `!countsetup`. If you accidentally break
the streak, you can pay JC to save it instead of starting over.

## 🎵 Music

Join a voice channel, then:
| Command | What it does |
|---|---|
| `/play <song or link>` | Play (or queue) a song |
| `/skip` | Skip the current song |
| `/pause` / `/resume` | Pause or resume playback |
| `/queue` | See what's queued up |
| `/nowplaying` | Show the current track |
| `/volume <0-100>` | Adjust the volume |
| `/stop` | Stop and disconnect Jarvis from voice |
| `/controls` | Open a control panel with buttons instead of typing commands |

## 💬 Feedback

Found a bug or have an idea? Jarvis wants to hear it:
- `/bugreport` — report something broken
- `/feedback` — suggest an improvement

If your suggestion or bug report gets accepted, you'll earn 50 JC as a thank-you.

---

## 📜 The Rules (please read)

- Don't spam commands — flooding Jarvis with too many commands too quickly will get you
  **temporarily timed out** from using the bot.
- There's a short cooldown between commands, so give it a second between messages.
- Being banned from Jarvis (globally or just on this server) means none of its commands will
  work for you — you'll get a one-time DM letting you know if this happens.
- Full command list any time: run `/help` for an interactive, categorized menu.
